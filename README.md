# Cine

Sistema de gestión de reservas de asientos para cine.

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/DevsataAlbo/cine.git
cd cine
```

## Ejecutar en local

Abre el archivo `index.html` directamente en tu navegador, o usa Live Server en VS Code.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3
- Font Awesome 6.4
